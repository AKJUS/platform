## Tools and Scripts

### Package Management

```bash
# Install dependencies
bun install

# Add a new dependency
bun add <package> --workspace=@tuturuuu/<scope>

# Update dependencies
bun update-all
```

### Development Commands

```bash
# Start all apps
bun dev

# Start specific app
bun --filter @tuturuuu/web dev

# Run tests
bun test

# Build all packages
bun run build
```

### Database Commands

```bash
# Start Supabase
bun sb:start

# Apply migrations
bun sb:push

# Generate types
bun sb:typegen

# Reset database
bun sb:reset
```

## Troubleshooting

### Common Issues

1. **Build Failures**: Check TypeScript errors and missing dependencies
2. **Database Issues**: Ensure Supabase is running and migrations are applied
3. **Test Failures**: Check for environment setup and mock configurations
4. **Performance Issues**: Use React DevTools and bundle analyzers

### Getting Help

- Check existing documentation
- Search GitHub issues
- Ask in team channels
- Create detailed bug reports
