# @tuturuuu/ai

AI utilities and helpers for Tuturuuu Platform.

## Installation

```bash
npm install @tuturuuu/ai
# or
yarn add @tuturuuu/ai
# or
bun add @tuturuuu/ai
```

## Development

```bash
# Install dependencies
bun install

# Run tests
bun test
```
