# @tuturuuu/supabase

Supabase client and utilities for Tuturuuu Platform.

## Installation

```bash
npm install @tuturuuu/supabase
# or
yarn add @tuturuuu/supabase
# or
bun add @tuturuuu/supabase
```

## Development

```bash
# Install dependencies
bun install

# Run tests
bun test
```
